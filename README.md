# Ecommerce-excel-dashboard
Interactive Excel-based e-commerce dashboard with KPI tracking, pivot analysis, solver optimization, and VBA automation
This project is a fully interactive Excel-based e-commerce performance, analytics, and automation system built for a Home & Kitchen product business.
It combines data modeling, pivot analytics, dashboards, optimization, and VBA automation into one end-to-end workflow.

The file simulates how a real e-commerce team would track sales, inventory, profitability, and operational decisions inside Excel.
